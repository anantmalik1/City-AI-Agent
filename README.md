## 🌌 What is NagarDoot?

<table>
<tr>
<td width="60%">

**NagarDoot** (नगरदूत — *Messenger of the City*) is a **fully AI-powered city assistant** built with LangChain + Mistral AI. It gives you real-time weather, live news, and intelligent answers about any Indian city — wrapped in a jaw-dropping **3D snow-animated Streamlit UI**.

> *"Not just a chatbot. A living city intelligence."*

## ⚡ Features

<div align="center">

| 🌟 Feature | 📝 Description |
|:---:|:---|
| ❄️ **Live Snow Canvas** | 220 physics-based snowflakes with 3D radial glow rendered on HTML5 Canvas |
| 🌡️ **Real-Time Temperatures** | Left sidebar shows live temperatures for 8+ Indian cities via OpenWeather API |
| 🤖 **LangChain Agent** | Mistral-small-2506 LLM with tool-calling for Weather + News |
| 📰 **Live News** | Tavily search integration — fetches top 3 news stories per city query |
| 💎 **3D Glassmorphic UI** | CSS `rotateX/Y`, floating animation, frosted glass panels |
| ✨ **Animated Sidebar Line** | Scrolling neon accent line on sidebar left edge |
| 🎯 **Auto-Clear Input** | Input box auto-clears after every message send |
| 🔄 **One-Click Refresh** | Refresh all city temperatures in one click |

## 🚀 Quick Start

### 1️⃣ Clone the repo
```bash
git clone https://github.com/anantmalik125/NagarDoot.git
cd NagarDoot
```

### 2️⃣ Install dependencies
```bash
pip install streamlit langchain-mistralai langchain tavily-python \
            python-dotenv requests rich
```

### 3️⃣ Set up your `.env` file
```env
OPENWEATHER_API_KEY=your_openweather_key
MISTRAL_API_KEY=your_mistral_key
TAVILY_API_KEY=your_tavily_key
```

| Key | Where to get it | Free? |
|-----|----------------|-------|
| `OPENWEATHER_API_KEY` | [openweathermap.org](https://openweathermap.org/api) | ✅ Yes |
| `MISTRAL_API_KEY` | [console.mistral.ai](https://console.mistral.ai) | ✅ Yes |
| `TAVILY_API_KEY` | [tavily.com](https://tavily.com) | ✅ Yes |

### 4️⃣ Run it!
```bash
streamlit run app.py
```

Open your browser at **`http://localhost:8501`** 🎉
